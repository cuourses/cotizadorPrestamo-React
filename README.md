# Cotizador Prestamo

## Descripción

Este proyecto consiste en una aplicación web desarrollada con React que permite calcular los pagos mensuales y el total a pagar en un préstamo financiero. La aplicación utiliza componentes personalizados como el encabezado y botones, así como funciones auxiliares para realizar los cálculos necesarios.

## Requisitos

Ejecutar el comando en la raiz del proyecto para la instalacion de dependencias

```
npm i
```

## Instrucciones de uso

Para acceder a la página, simplemente corr el comando

```
npm run dev
```

para visualizar cambios agregados.
